# react-starter #
A production ready react boilerplate.

### Get it ###
`git clone https://github.com/nunisa/react-starter.git`

### Usage ###
After cloning the repo, do the following
1. Install dependencies

    `npm i`
2. Run development server

    `npm start`
3. Run production build

    `npm run build`
4. Run jest tests

    `npm test`
